# assets/images

Static images used by the site (illustrations, Open Graph cover images, screenshots referenced in tutorials, etc).

- `hero-illustration.svg` — decorative sample illustration included with the project.

Add your own photos/screenshots here and reference them with a relative path, e.g.:

```html
<img src="assets/images/your-file.jpg" alt="Describe the image">
```

Prefer `.svg` for icons/illustrations (crisp at any size, tiny file size) and `.webp` or `.jpg` for photos.
