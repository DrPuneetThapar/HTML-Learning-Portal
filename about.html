About the HTML Learning Portal

A self-contained, classroom-ready platform for teaching HTML to undergraduates — built entirely with static files, so it can be hosted anywhere with no backend.

What's inside

  • 16 topics covering everything from how the Internet works to HTML best practices.
  • 152+ runnable, copyable code examples across the tutorials.
  • A Monaco-powered live editor (the same engine behind VS Code) in the Playground and Practice pages.
  • 48 quiz questions with instant feedback, plus a mixed-topic challenge mode.
  • A searchable reference of 76 common HTML tags.
  • Progress tracking, badges, and bookmarks that persist in your browser.

How progress tracking works

Your completed topics, quiz scores, bookmarks, and notes are saved to your browser's local storage under a single key. Nothing is sent to a server — it's private to your device and this browser. Clearing your browser data will reset your progress.

Technology

HTML5 & CSS3

Semantic, hand-written markup and a custom design system — no framework CSS overrides needed beyond Bootstrap's grid.

Vanilla JavaScript

No build step, no framework — plain ES6+ modules loaded directly by the browser.

Bootstrap 5

Grid, navbar, and responsive utilities, layered under a custom visual theme.

Font Awesome

Iconography throughout the navigation, feature cards, and toolbars.

Monaco Editor

The same code-editing engine as VS Code, loaded from CDN for the Playground and Practice pages.

JSON data files

All tutorial, quiz, and reference content lives in /data, decoupled from the page markup.

Using this in a classroom

Host the folder on any static file server (GitHub Pages, Netlify, a school's web server, or even a local python -m http.server). Every page works standalone — link students directly to a specific module with tutorial.html?topic=lists, for example.
